# Personal Profile Page

## About the Project

This is a simple personal profile webpage that I created while learning HTML. The main goal of this project was to practice the basic structure of an HTML document and understand how different HTML elements work together.

I created a webpage that includes a short introduction about myself, my skills, and other basic information. Although it is a simple project, it helped me become more comfortable with writing clean and organized HTML code.

## Technologies Used

* HTML5

## What I Practiced

* Creating the basic HTML structure
* Using headings and paragraphs
* Creating unordered lists
* Organizing content into sections
* Writing clean and readable HTML

* Add CSS for styling
* Make the page responsive
